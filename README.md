Projeto de Acessibilidade Web - Atividade 4

Este projeto foi desenvolvido para implementar recursos de acessibilidade em um site, seguindo as boas práticas WCAG e garantindo inclusão para todos os usuários.

Objetivo do Projeto

Tornar o site acessível, oferecendo:

Navegação fácil e intuitiva

Leitura confortável para diferentes perfis de usuários

Contraste adequado e modos visuais alternativos

Compatibilidade com teclado e leitores de tela

Recursos Implementados
Botão de Acessibilidade Global

O botão está localizado no topo de todas as páginas, garantindo acesso rápido e uniforme aos recursos de acessibilidade.

Funções em JavaScript

Aumentar fonte: aumenta o tamanho do texto em toda a página

Diminuir fonte: reduz o tamanho do texto

Ativar alto contraste: aplica contraste elevado para melhor visibilidade

Ativar modo escuro: muda a aparência da página para reduzir fadiga ocular

Restaurar configurações padrão: retorna todas as opções às configurações originais

Classes CSS para Acessibilidade

.high-contrast: contraste elevado

.dark-mode: modo escuro

Ajustes de fontes utilizando variáveis CSS no :root para consistência

Estrutura HTML Reorganizada

Uso correto de elementos semânticos: header, nav, main, section, footer

Textos alternativos (alt) em todas as imagens

aria-label aplicado a botões

Navegação por teclado totalmente funcional

Diferenciais Visuais

Sistema de cores acessível e consistente

Fontes ajustáveis sem quebrar o layout

Modos visuais alternativos que melhoram a experiência do usuário

Considerações Finais

O projeto demonstra como melhorias simples podem tornar a web mais inclusiva, garantindo uma experiência de navegação confortável e acessível para todos os usuários.
